#mip

Usage: /usr/bin/mip

Description: List all the interfaces in the system and their IP addresses

Sample use:
<pre>
$ mip <br>
No  | Interface            | IP              
----+----------------------+---------------- 
   1| lo                   | 127.0.0.1       
   2| enp9s0.42            | -               
   3| enp9s0               | -               
   4| wlx00e62010094b      | -               
   5| vmnet1               | 172.16.106.1    
   6| vmnet8               | 172.16.165.1    
   7| wlp7s0               | 10.10.10.196    
</pre>
